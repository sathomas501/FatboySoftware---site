# FB Retirement Planner Website

Marketing website for FB Retirement Planner - a financial and retirement planning application.

## Structure

- `index.md` - Main landing page
- `screenshots.md` - Screenshots gallery
- `assets/images/` - Screenshot images

## GitHub Pages Setup

This site is designed to work with GitHub Pages using Jekyll.
